Segurança e Privacidade no Chat GPT4 via SkyDeck Genstudio

GPT-4 é incrível. Mas o GPT-4 no ChatGPT não é privado. Nós resolvemos este problema. O GPT-4 no SkyDeck AI oferece a você um chat privado.

Quando você usa o ChatGPT para consumidores, mesmo que tenha pago por ele, suas discussões não permanecem privadas. Elas podem ser usadas pela OpenAI para treinar seus modelos. É por isso que muitas organizações proibiram o uso do ChatGPT.

Mas se você usar sua chave de API da OpenAI no SkyDeck para habilitar chat e ferramentas inteligentes em nosso espaço de trabalho GenStudio, essas conversas não são usadas pela OpenAI e permanecem privadas. Isso é verdade para todas as classes de chaves de API da OpenAI. Este é um caminho para o uso seguro dos avançados modelos GPT-4 e GPT-3.5 da OpenAI.

Uso da API da OpenAI

Uma das principais vantagens de usar sua chave de API OpenAI no SkyDeck AI é que isso permite acesso privado aos sistemas da OpenAI. Se alguém usar os serviços gratuitos ou pagos do ChatGPT, a OpenAI não faz promessas sobre a segurança dos dados. Mas se alguém usar uma chave de API OpenAI no Control Center, toda a interação de chat pelos seus usuários no GenStudio permanece privada.

Isso é o queOpenAI escrevesobre o assunto:

“Os dados enviados através da API OpenAI não são usados para treinar modelos da OpenAI ou melhorar a oferta de serviços da OpenAI. Dados enviados através de serviços de consumo que não são API, como ChatGPT ou DALL·E, podem ser usados para melhorar nossos modelos.”
